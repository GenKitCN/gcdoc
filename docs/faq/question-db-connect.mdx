## 下载Navicat

登入Navicat[官网](https://www.navicat.com.cn/download/navicat-premium)

选择你系统版本并下载，根据安装程序走即可

## 连接
点击左上角连接，在列表中选择MongoDB
![image](https://user-images.githubusercontent.com/106385654/180144221-2df6f352-2a5f-4dc5-8373-f0d055807c29.png)
填写连接名，其余默认即可(在本地使用的话)

当然如果你会自己捣鼓，其他的都可能需要修改，但这里不作进一步解释

![image](https://user-images.githubusercontent.com/106385654/180144338-05fd1fae-5ebd-4ebb-add8-9ada90f342f5.png)

按下确定后，开启MongoDB，双击Navicat左侧你刚才填写的连接名，就可连接到服务器(看到下面Grasscutter容器后即为成功)
![image](https://user-images.githubusercontent.com/106385654/180144825-bc49f691-352f-489f-ba70-4165ae754d4e.png)

我们编辑游戏内文件都主要使用集合选项，双击展开即可进行查看

## 无限期试用(Windows)
新建bat文件，点击编辑，键入以下代码并保存
```bash
@echo off
echo Delete HKEY_CURRENT_USER\Software\PremiumSoft\NavicatPremium\Registration[version and language]
for /f %%i in ('"REG QUERY "HKEY_CURRENT_USER\Software\PremiumSoft\NavicatPremium" /s | findstr /L Registration"') do (
    reg delete %%i /va /f
)
echo.
echo Delete Info folder under HKEY_CURRENT_USER\Software\Classes\CLSID
for /f %%i in ('"REG QUERY "HKEY_CURRENT_USER\Software\Classes\CLSID" /s | findstr /E Info"') do (
    reg delete %%i /va /f
)
echo.
echo finish
pause
```

运行后即可续期
